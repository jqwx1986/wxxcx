## 如果 Git 命令不可用

Git 可能没有添加到系统 PATH，有两种解决方法：

### 解决方法1：重启电脑
重启后 Git 命令会自动可用。

### 解决方法2：添加到 PATH（临时）
在 PowerShell 中执行：
```powershell
$env:Path += ";C:\Program Files\Git\bin"
```

或者找到 Git 安装位置，使用完整路径：
```powershell
"C:\你的Git安装路径\bin\git.exe" init
```

## 还没有 GitHub Token？

### 创建 Token 步骤：

1. **访问 Token 页面**
   https://github.com/settings/tokens

2. **生成新 Token**
   - 点击 `Generate new token` → `Generate new token (classic)`

3. **配置 Token**
   - Note：`Git Upload`
   - Expiration：`90 days` 或 `No expiration`
   - Select scopes：✅ 勾选 `repo`

4. **生成并复制**
   - 点击 `Generate token`
   - ⚠️ **立即复制 Token**（只显示一次）
